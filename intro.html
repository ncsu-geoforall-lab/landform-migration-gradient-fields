Jockey’s Ridge dune

Created in GRASS GIS Animation Tool

Current techniques used to study landform migration have limitations. Jockey’s Ridge dune (Nags Head, The Outer Banks, NC) is moving and changing. Animation in 2D and 3D. Not quantitative.

Shoreline transects

We study evolving shorelines using transects. Transects divide shoreline and specify areas where we measure the changes. There are very good algorithms for creating transects, for example in GRASS GIS. However, transects for one set of shorelines might not work when shoreline changes significantly.

Connecting contours

Shorelines and contours are similar. To relate two different shorelines or contours we can draw the connections manually. This is a lot of work and it's impossible to reproduce.
